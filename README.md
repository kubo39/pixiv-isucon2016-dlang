# Pixiv 社内ISUCON 2016 D言語実装

dlang app for https://github.com/catatsuy/private-isu

## (WIP) セットアップ

```
$ cd private_isu/webapp
$ git clone https://github.com/kubo39/pixiv-isucon2016-dlang dlang
$ cd dlang
```

## ビルド

```console
$ dub run
```
